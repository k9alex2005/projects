# Personal Portfolio Website

## Description

This project is a simple personal portfolio website that presents information about me, including my background, skills, hobbies, and coursework.

The goal of the project was to practice building and structuring a responsive front-end webpage using core web technologies while keeping the design clean and easy to navigate.

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap (Grid system for layout)

## Features

* Structured sections for personal information, skills, hobbies, and courses
* Responsive layout using Bootstrap grid
* Basic client-side interactivity implemented with JavaScript

## Running the Project

1. Clone the repository
2. Open `index.html` in a web browser

## Purpose

This project was created as practice for:

* Structuring web pages with HTML
* Styling layouts using CSS
* Implementing simple client-side interactivity with JavaScript
* Using Bootstrap’s grid system to create responsive layouts
